# editorconfig_gen

### Supported languages:

- Ruby
- Lua
- Python
- Julia
- JavaScript
- TypeScript
- Haskell
- Idris
- Agda
- StandardML
- OCaml
- Go (make)
- Zig
- Rust
- C (make)
- C++ (make)
- C#
- F#
- Kotlin (gradle)
- Java (gradle)
- Scala (sbt)
- Swift
- Objective-C

### Supported environments:

- HTML, CSS & JS
- Ruby on Rails
- Reactjs
- C# with XAML

### Usage:

```
$ ./bin/editorconfig.rb --help
```
